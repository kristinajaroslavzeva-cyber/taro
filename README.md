Политика конфиденциальности

Настоящая Политика конфиденциальности описывает порядок сбора, использования и хранения информации в приложении «Оракул Таро».

1. Общие положения

Приложение уважает право пользователей на конфиденциальность и обрабатывает данные в минимально необходимом объёме, исключительно для обеспечения работы функционала приложения.

2. Какие данные могут собираться
2.1 Данные при авторизации

При входе через сторонние сервисы (например, Google) приложение может получать:

идентификатор аккаунта;

имя профиля;

адрес электронной почты.

Эти данные используются только для авторизации пользователя и управления учётной записью.

2.2 Пользовательский контент

Вопросы, расклады и результаты интерпретаций:

не используются для идентификации личности;

не передаются третьим лицам;

могут храниться локально на устройстве пользователя.

3. Использование данных

Собранные данные используются исключительно для:

обеспечения работы приложения;

авторизации пользователя;

улучшения стабильности и качества сервиса.

Приложение не использует данные для рекламы и не передаёт их третьим лицам.

4. Передача данных третьим лицам

Приложение не продаёт и не передаёт персональные данные пользователей третьим лицам, за исключением случаев, предусмотренных действующим законодательством.

5. Хранение и защита данных

Мы принимаем разумные меры для защиты данных пользователей от несанкционированного доступа, изменения или удаления.

6. Права пользователя

Пользователь вправе:

прекратить использование приложения;

удалить учётную запись (если применимо);

отказаться от авторизации через сторонние сервисы.

7. Изменения политики

Политика конфиденциальности может обновляться. Актуальная версия всегда доступна в приложении.

8. Контактная информация

По вопросам, связанным с конфиденциальностью данных, пользователь может связаться с разработчиком по контактам, указанным в магазине приложений.

9. Удаление данных
Вы можете в любой момент удалить Приложение со своего устройства. Если вы хотите, чтобы вся история ваших диалогов была удалена с наших серверов, пожалуйста, свяжитесь с нами по электронной почте molchan130786@gmail.com
